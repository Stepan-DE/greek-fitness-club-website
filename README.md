# Сайт для фитнес клуба премиум сегмента в стилистике Древней Греции

**Это мой первый учебный проект по верстке сайта**

## Концепт:
В колледже где я учусь есть предмет web-разработки и недавно мы реализовывали проект, мне выпала тема спорта и я решил сделать сайт для фитнес клуба премиум сегмента с тематикой Древней Греции. Название "Telos" (греч. "Цель, совершенство") отражает философию клуба — помощь в достижении физического и ментального совершенства.

Можно ознакомить с сайтом запустив его у себя, посмотреть видео в файле greek-site-privie.mp4 или же по ссылке в VK видео: https://vkvideo.ru/video-235877670_456239017?list=ln-neOk763O3Wgvlml4xr

## Цель проекта:
Создать современный, визуально эффектный лендинг для премиального фитнес-клуба, используя тематику Древней Греции для передачи ощущения элитности, силы и традиций.

## Ключевые особенности и реализация:
1. **Закреплённая шапка** сайта с анимированной навигицией
2. **Кастомные шрифты и иконки** для продвинутого дизайна
3. **CSS-анимации** элементов при взаимодействии:
   - Плавное изменение цвета кнопок при наведении
   - Плавная прокрутка к якорям при переходе по ссылкам
   - Анимированные блоки с эффектом подсветки в разделе "Направления"
   - Плавное масштабирование фотографий в разделе "Галерея"
   - Анимация блоков и кнопок в разделе "Членство"
   - Подсветка активного поля в форме "Контакты"
    
## Как запустить проект:
**Рекомендуется запустить сайт у себя локально, так как превью не способно полностью передать весь опыт использования**
1. Клонируйте данный репозиторий 
'bash
git clone https://github.com/Stepan-DE/greek-fitness-club-website.git
'
2. Откройта файл 'index.html' в любом современном браузере (Firefox, Google, Yandex и тд.)

## Этапы дальнейшего развития сайта: 
Сайт является только макетом для отрботки с версткой и требует не малых улучшений, таких как:
- оптимизация кода: имеются повторяющиеся стили которые были нужны для экспериментов с дизайном, код следует
- сделай адаптивную масштабируемость: сайт написан под конкретные параметры экрана и при уменьшении масштаба может ломаться, следует добавить нефиксированные величины и сделать адаптацию на мобильные устройства и мониторы низкого разрешения
- добавление JS для более продвинутых анимаций и (напиши что еще можео добавить)
- создание серверной части сайта и подключение Базы данных для рабочей регистрации
- добавить дополнительные страницы для создания функционала кнопкам "Подробнее"
